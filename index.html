    • I re-architected an internal legacy application in Typescript by separating configurations into AWS AppConfig from business logic in AWS Lambda which reduced deployment time for configuration changes from 1 week to 1 day. I wrote unit tests with 100% coverage for these changes using Jest.
    • I added local deployment functionality using Typescript allowing teammates to easily deploy local changes to personal accounts through the command line for testing configuration changes in JSON.

    • Using the account creation pipeline and script, I swiftly created 20 pre-production accounts for a 1:1 mapping with each production region. I onboarded our services and its dependencies to those accounts so that we can accurately test in those regions before releasing changes to production.

    Skill Swap

    A skill trading web application where users can share knowledge with the world, in exchange for knowledge from others. Users can create an account by logging in using their google email login, using the built-in Google Login API (profile information such as name, email, and user icons are pulled from the user's google login). Users may add skills to their profile, complete with descriptions, and expected time commitments for instruction. They can search by username or skill name in the provided search bar and click on a result to go to a user's page. And finally, users may initiate skill swaps with each other and trade knowledge!

    Tech stack:
    Front-end - React/JS using Material UI and SASS frameworks
    Back-end - Python/Flask/PostgresSQL
    Outside API - Google Login API
